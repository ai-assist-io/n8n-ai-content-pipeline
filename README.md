# AI Telegram Content Automation System

AI-powered Telegram automation workflow built with n8n.

## Features
- AI text generation
- AI image generation
- Telegram Bot integration
- Approval/Decline moderation system
- Automated publishing to Telegram channel
- Human-in-the-loop workflow

## Tech Stack
- n8n
- OpenAI API
- Telegram Bot API
- Workflow Automation

## Workflow Logic
1. User sends prompt via Telegram
2. AI generates text and image
3. Preview sent for moderation
4. User approves or declines
5. Approved content gets published automatically
